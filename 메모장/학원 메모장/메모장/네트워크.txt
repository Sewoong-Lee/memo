subnet mask : 네트워크 주소
a클래스 255.0.0.0
b클래스 255.255.0.0
c클래스 255.255.255.0

ip 주소로 클래스 구별
a클래스 0~127 (아이피 첫 숫자) : 비트로 따지면 0으로 시작
b클래스 128~191 : 10으로 시작
c클래스 192~223 : 110으로 시작
ip 192.168.0.100
ip 192.168.0.200
ip 192.168.1.100
ip 192.168.1.200

cidr표기법 : 네트워크 주소 길이로 표기
ip 192.168.0.100 / 24
ip 192.168.0.200 /24
ip 192.168.1.100 / 24
ip 192.168.1.200 /24

서브네팅 예제
192.168.10.0 /24  -> 네트워크 주소가 3개 필요 서브네팅을 어떻게?
192.168.10.0 /26
1)192.168.10.0 /26 : 00000000 ~00111111 : 64개
2)192.168.10.64 /26 : 01000000 ~01111111 : 64개
3)192.168.10.128 /26 : 10000000 ~10111111 : 64개
4)192.168.10.192 /26 : 11000000 ~11111111 : 64개

---라우팅 세팅--- 게으트웨이 ip설정
라우팅 클릭 cli -> no -> 엔터 2번 -> 
1)사용자 모드 : 장비 상태 확인 등
   Router> enable  관리자 모드로
2)관리자 모드 : 정보 확인, 저장, 삭제,초기화
   Router# config terminal 글로벌 모드로
3)글로벌 모드 : 상세 설정 
   Router(config)# interface gigabitEthernet 0/0/0
   Router(config-if)# ip address 192.168.0.1 255.255.255.0
   (2번째꺼)Router(config-if)# ip address 192.168.1.1 255.255.255.0
   no shutdown (장비 켜기)
   exit 단계별 빠져나가기

--선생님 해석
라우터에 라우팅 테이블 만들기(경로설정)
1)192.168.0.0/24의 라우터
--192.168.1.0 네트워크 주소로 가려면 10.0.0.2 ip로 가야 한다
Router(config)#ip route 192.168.1.0 255.255.255.0 10.0.0.2
--128.0.0.0/16 네트워크 주소로 가려면 10.0.0.2 ip로 가야 한다
ip route 128.0.0.0 255.255.0.0 10.0.0.2

2)192.168.1.0/24의 라우터
--192.168.0.0 네트워크 주소로 가려면 10.0.0.1 ip로 가야 한다
ip route 192.168.0.0 255.255.255.0 10.0.0.1
--128.0.0.0/16 네트워크 주소로 가려면 20.0.0.2 ip로 가야 한다
ip route 128.0.0.0 255.255.0.0 20.0.0.2

3)128.0.0.0/16의 라우터
--192.168.1.0/24 네트워크 주소로 가려면 20.0.0.1 ip로 가야 한다
ip route 192.168.1.0 255.255.255.0 20.0.0.1
--192.168.0.0/24 네트워크 주소로 가려면 20.0.0.1 ip로 가야 한다
ip route 192.168.0.0 255.255.255.0 20.0.0.1
